# Space Simulator – Solar System & Milky Way

An interactive 3D space simulator built in the browser.

## Features

- *Solar System View*
  - Sun and 8 planets with orbit paths
  - Labels for planets
  - Asteroid belt

- *Galaxy View*
  - Milky Way–style spiral galaxy
  - Central supermassive black hole

- *Black Hole Focus*
  - Camera focuses on the black hole
  - Accretion disk animation

## Controls

- Drag to rotate the camera  
- Scroll to zoom  
- Buttons to switch:
  - Solar System
  - Galaxy View
  - Focus Black Hole
- Time control: pause / play and speed slider  
- Toggles:
  - Show orbit paths
  - Show labels
  - Show starfield

## Tech Stack

- WebGL / Three.js (or similar 3D library, generated in Replit)
- HTML, CSS, TypeScript/JavaScript
